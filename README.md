# Intune Scripts

PowerShell scripts designed for use with Microsoft Intune to support application deployments, compliance policies, and endpoint remediation.

## 📌 Purpose
This repository contains scripts used in Intune environments to automate device configuration, enforce compliance, and improve visibility across managed endpoints.

## 🛠 Scripts Included
- Detection scripts for Win32 applications
- Remediation scripts for compliance issues
- Configuration and deployment helpers
- Reporting scripts for endpoint state

## 🚀 Usage
Scripts can be used in:
- Intune Win32 app deployments (install/detection rules)
- Proactive remediation scripts
- Manual execution for testing

## ⚠️ Notes
- Ensure scripts are tested before deployment to production devices
- Some scripts may require system context within Intune
- Logging/output may need to be adapted for Intune reporting

## 🎯 Goal
To standardise and streamline endpoint management through reusable and reliable automation in Intune.
